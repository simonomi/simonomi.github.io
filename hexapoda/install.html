cargo

currently, the recommended way to install hexapoda is through cargo, using one of the following commands:

cargo install hexapoda
cargo binstall hexapoda

cargo-binstall will use pre-compiled binaries if available (~2s vs ~20s on my machine), but it must be installed separately on top of cargo itself

pre-built binaries

you can manually download binaries at the GitHub releases page, which come with shell completions and man pages
